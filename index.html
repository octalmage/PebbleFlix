PebbleFlix

Control Netflix with your watch (Pebble).

Instructions

PebbleFlix has two parts, the watch app and the server app.

The server app is a Mac app. First download and run PebbleFlixServer.zip.

The watch app is in the Pebble Store, search for "PebbleFlix".

With the server app running and the watch app installed, launch the watch app.

The watch app should connect automatically then you're ready to control Netflix!
